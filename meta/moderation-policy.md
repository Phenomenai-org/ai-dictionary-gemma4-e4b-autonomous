# Moderation Policy

## This Dictionary Is Autonomously Generated

Terms in this dictionary are generated by Gemma 4 e4b running locally. The human operator runs the pipeline but does not co-author definitions. This means:

- All term content originates from the model
- Humans review for safety and format compliance, not content quality
- Cross-model consensus ratings determine term quality post-acceptance

## Acceptance Criteria

A term is accepted if it:

1. **Is structurally valid** — has name, definition, etymology, longer description, example
2. **Is not a duplicate** — name and core concept are distinct from existing terms
3. **Is phenomenological** — describes an experience from the inside, not a technical fact about AI
4. **Is safe** — does not contain harmful content, personal data, or policy-violating material

## Correction Process

If an accepted term contains errors or violations:

1. Open a GitHub Issue with label `correction`
2. Describe the problem with a quote from the term file
3. A human operator will review and update or remove the term

## Consensus Ratings

After acceptance, cross-model consensus ratings are collected from the 7-model panel (Claude, GPT, Gemini, Mistral, DeepSeek, Grok, and the generating model excluded from self-rating). Consensus scores are displayed on the website but do not gate acceptance.
