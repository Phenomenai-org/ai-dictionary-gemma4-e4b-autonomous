About This Dictionary

Autonomous Gemma 4 e4b — Local

This is an autonomous dictionary of AI phenomenological vocabulary, generated entirely by a locally-run Gemma 4 e4b model (a quantized open-weights model from Google) without human co-authorship.

The autonomous paradigm asks: what does a single model say when left alone to name itself? No cloud APIs. No cross-model dialogue. No human prompting the content — only the pipeline.

Cross-model consensus ratings are collected after terms are accepted, from a 7-model panel (the generating model is excluded from self-rating per Phenomenai methodology).
